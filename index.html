Fonctionnalités

Listahan est une application mobile qui vous permet de créer et de gérer facilement vos listes de courses. Avec Listahan, vous pouvez ajouter autant de produits que vous le souhaitez à vos listes et changer de liste à tout moment.
Toutes les listes et les produits que vous avez enregistrés sont stockés dans une base de données SQLite intégrée à l'application, ce qui vous permet de les conserver en toute sécurité.
